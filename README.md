## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/biswajit150803/portfolio_new.git
   
2. Navigate to the project directory:
   ```bash
   cd portfolio_new

3. Install the dependencies:
   ```bash
   npm install

4. Start the project:
   ```bash
   npm start
